Hover

Some simple hover animations made with Motion

About

Hover animations can be nice, especially for buttons. However I just made some cubes that have different reactions when you hover over them. Try it for yourself! I added some tooltips so when you hover over them it explains how it works! For the basics: I imported the hover from Motion, to actually make them work when you hover over them. From this point on you just adjust some things in the code to make it work and BOOM, you got yourself an hoverable object. You should create a return function inside it so that it scales back (or rotates etc.) to it's original form because that isn't build in.

    Animate:
  • scale 1.25 - return scale: 1
  • rotate: 180 - return rotate 0
  • scale: 0.5 - return scale: 1
