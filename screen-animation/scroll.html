About

This was my second Motion project, still getting started with everything. Most of this code is copy + paste (Retype everything) from the Motion docs. Best way to learn something is to write about it so lets get started.

For the main-div with all the sub-divs I made simple classes corresponding to their function. 

            
    <div class="scroll-menu">
        <section class="scroll-item">Placeholder 1</section>
        <section class="scroll-item">Placeholder 2</section>
        <section class="scroll-item">Placeholder 3</section>
        <section class="scroll-item">Placeholder 4</section>
    </div>

The goal of the scroll animation is to animate the elements when they get scrolled in and out of the view of the user. When the user scrolls over the element the animation gets played and it gets loaded in. When the user scrolls past the element it gets hidden again. This is in my opinion a very useful way to simplify animations and make the website feel a little bit more alive.
