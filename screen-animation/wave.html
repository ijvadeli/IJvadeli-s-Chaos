Wavy Text

Small CSS and JS project

About

This is as the title suggests wavy text. It's not super complicated, because the animation itself is CSS keyframes. The JavaScript looks at all the characters inside the text and seperates them. This makes the animation play for all the characters seperately resulting in this wave like pattern. I'm planning on adding sliders someday, so you can mess around with the code and maybe download it as a snippet to use for your own projects. For now, you can just download the file and have a look at the code yourself!
